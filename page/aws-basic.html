개발자와 스타트업을 위한 AWS Basic

4주간, AWS 공인 테크니컬 트레이너 출신 강사에게 배워보세요 !

신청이 마감됐어요 😑

과정 특징


전, AWS 시니어트레이너의 강의

정호영 마스터는 컴퓨터 공학박사이며, LG전자, NAVER에서 개발자로 근무한 경험이 있으며, 아마존의 AWS공인 트레이너로 일한경력이 있는 클라우드 전문가 입니다. AWS 시니어트레이너로 일하며 수많은 기업의 컨설팅과 개발문제를 경험했습니다. 또한 교육전문가로써 NHN NEXT와 코드스쿼드에서 데이터베이스, 알고리듬, AWS 등의 교육을 맡고 있습니다


실용적인 커리큘럼

강사의 풍부한 AWS경험으로부터 실무에서 활용가능한 커리큘럼으로 구성됐습니다.
'AWS시작하기'부터 'Computing', 'Storage', '배포 및 운영'까지 실무적인 내용을 강의와 실습을 통해서 배울 수 있습니다.


실전에 필요한 내용

교육중에는 국내외의 실제 AWS사용사례에 대해서 알 수 있습니다. 이를 통해서 실전에서 어떻게 사용할 수 있는지 많은 힌트를 얻을 수 있습니다. 다양한 겉핥기 수업이 아닌 실전에 필요한 핵심 서비스 위주로 깊이 있는 내용을 실습할 수 있습니다.

커리큘럼

개발자와 스타트업을 위해서 가장 필요한 내용을 4주로 구성했습니다. 중간에 과제도 있으며 1:1리뷰를 받을 수 있습니다.

기간주제내용
1주차

AWS 시작하기

  • IAM으로 안전한 서비스 시작하기
  • VPC와 AWS 네트워크 기초
  • EC2, EBS, T2 및 컴퓨팅 인스턴스의 이해
  • S3 와 다양한 저장장치들의 활용
2주차

Computing

  • CLI 사용법
  • SDK 사용법
  • CloudWatch를 이용한 모니터링
  • Lambda
  • 통합 실습
3주차

Storage

  • S3 응용
  • EBS 더 알아보기
  • RDS 사용해 보기
  • 프로그래밍 실습: 서비스 연동 어플리케이션 구축하기
4주차

배포 및 운영

  • AMI, ELB 와 Route53 이용한 배포 및 운영 실습
  • Elastic Beanstalk을 이용한 간단 배포 실습
  • Code Deploy 등을 이용한 Github 배포 실습

어떤분들에게 필요한 강의인가요?

'AWS를 도입하려는 스타트업 개발자', '클라우드를 배우고자 하는 취업 준비생', '일반 개발자' 에게 필요한 강의 입니다

강사의 메시지

AWS에는 수많은 서비스들이 있습니다. 제한된 시간에 모든 걸 배울 수는 없지만 서비스 운영에 필요한 필수적인 내용을 과정에 담았습니다. 그리고 과정이후에는 여러분들이 AWS를 이해하는 것에 그치지 않고, AWS의 운영방법을 깨닫게 되실 겁니다.

'개발자와 스타트업을 위한 AWS기초'과정을 통해서 클라우드 기반의 서비스를 완전히 마스터해보세요!
