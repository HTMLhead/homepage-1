### 파일 디렉토리

`/content/recruit/jobPosition.mdx`

### 예시

```
---
templateKey: recruit_jobPositions
jobPositions:
  - category: 교육개발
    title: 웹 프론트엔드 교육 개발자
    content: 웹 프론트엔드 교육 개발자를 채용 정보
    editDate: "2022-04-21"

  - category: 교육개발
    title: 웹 백엔드 교육 개발자
    content: 웹 백엔드 교육 개발자를 채용 정보
    editDate: "2022-04-22"
---
```

### 항목별 설명

- jobPositions: 각 업무들을 보여주는 항목입니다.
  - category: 카테고리를 입력할 수 있습니다.
  - title: 업무명을 입력할 수 있습니다.
  - content: 업무의 주된 설명을 작성할 수 있습니다.
  - editDate: 수정된 날짜를 입력할 수 있습니다. 날짜만 입력하면 됩니다.
