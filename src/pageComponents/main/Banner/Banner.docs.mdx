### 파일 디렉토리

`/content/main/banner.mdx`

### 예시

```
---
templateKey: main_banner
title: 2023년 마스터즈 코스 절찬 모집중
description: 2022. 12. 00부터 12. 00 오후 1시까지
---

```

### 항목별 설명

- title: 배너에 입력될 제목 텍스트를 입력할 수 있습니다. 공백("")이라면 배너가 보여지지 않습니다. 아예 아무값도 주지 않게되시면 에러가납니다
- description: 배너에 입력될 설명 텍스트를 입력할 수 있습니다.
