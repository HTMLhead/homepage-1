### 파일 디렉토리

`/content/code-together/features.mdx`

### 예시

```
---
templateKey: codeTogether_features
features:
  - title: 함께하는 학습
    img: eduFeature1
    descriptions:
      - 동료들과 그룹을 지어 함께 학습
      - 주어진 미션을 같이 고민하며 함께 성장

  - title: 주제별 미션 해결
    img: eduFeature2
    descriptions:
      - 주제별 n개의 미션을 해결
      - 작은 미션을 자주 만들면서 제대로 학습

  - title: 학습을 돕는 활동과 콘텐츠
    img: eduFeature3
    descriptions:
      - 좋은 코드에 대해 함께 고민하는 코드리뷰
      - 필요한 지식을 탄탄히 학습할 수 있는 VOD
---
```

### 항목별 설명

- features: 코드 투게더의 특징에 대한 내용들을 보여주는 항목입니다.
  - title: 항목에 사용될 제목을 입력할 수 있습니다.
  - descriptions: 항목을 설명하는 문구들을 입력할 수 있습니다.
  - img: 카드에 보여지는 이미지를 입력할 수 있습니다.
