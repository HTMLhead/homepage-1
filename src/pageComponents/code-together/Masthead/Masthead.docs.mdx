### 파일 디렉토리

`/content/code-together/masthead.mdx`

### 예시

```
---
templateKey: codeTogether_masthead
title: 코드투게더
description: 코드투게더는 풀타임 수업 참여가 어려운 사람들을 대상으로, 다양한 수준의 과정들을 학습할 수 있는 코드스쿼드만의 교육 모델입니다. 다양한 미션들을 마주하며 프로그래밍에 대한 흥미를 키우고 개발 요구사항에 대응할 수 있는 능력을 기릅니다.
scheduledCourses:
  - title: 자바스크립트
    dueDate: 조기마감
    img: jsPart1
    path: /code-together/javascript

  - title: 자바스크립트
    dueDate: 5월 20일 오후 4시 마감
    img: jsPart2
    path: /code-together/javascript

  - title: 자바스크립트
    dueDate: 5월 20일 오후 4시 마감
    img: jsPart3
    path: /code-together/javascript
---
```

### 컬럼별 설명

- title: 코드투게더 페이지 헤더의 제목을 입력할 수 있습니다.
- description: 코드투게더 페이지 헤더의 내용을 입력할 수 있습니다.
- scheduleCourses: 현재 모집중인 과정들을 보여줍니다.
  - title: 과정의 제목을 입력할 수 있습니다.
  - dueDate: 모집 마감일시를 입력할 수 있습니다.
  - path: 클릭시 이동될 url을 입력할 수 있습니다.
  - img: 코스 이미지를 입력할 수 있습니다.
