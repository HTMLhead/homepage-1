➀ 2022 마스터즈 코스 접수 방법
  • 신청: 1차 테스트 문제를 풀어서 기한내에 제출하면 접수 완료(따로 접수 신청 없음)
  • 합격기준 : 1차와 2차 두 번의 테스트를 모두 통과해야 함
  • 모집인원: 한 클래스당 20-50명
➁ 1차 테스트 일정
  • 1차 테스트 방식: 공개된 문제를 풀어서 기한 내 제출
  • 1차 테스트 문제 공개 날짜: 2021년 12월 6일 월요일 오전 10시
  • 1차 테스트 문제 공개 장소: 문제 사이트 바로가기
  • 1차 테스트 제출 마감 날짜: 2021년 12월 8일 수요일 밤 10시
  • 1차 테스트 문제 제출 방법: 문제 공개 사이트에서 함께 안내
  • 1차 테스트 합격자 발표: 2021년 12월 10일 금요일 오전 11시 이메일로 안내
➂ 2차 테스트 일정
  • 2차 테스트 방식: 2차 문제 풀이 또는 1:N 면접 (온라인 또는 오프라인. 1차 합격 인원에 따라 다름)
  • 2차 테스트 날짜: 2021년 12월 13일/14일 택1 (참가 시간은 1차 테스트 합격자에게 별도 안내)
  • 최종 합격자 발표: 2021년 12월 16일 목요일 오전 11시 이메일로 안내 (불합격자에게도 이메일로 안내합니다)
