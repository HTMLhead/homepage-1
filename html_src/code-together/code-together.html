CODE TOGETHER

코드투게더는 여러 동료들과 ‘함께’ 소프트웨어를 학습하는 과정입니다.
교육 시간이 주 3일 / 평일 저녁 시간대에 진행되어, 풀타임 학습이 어려운 분들도 수강할 수 있습니다.
미션 실습과 문제 해결 중심으로 진행되며, 그 과정에서 학습을 도울 수 있는 영상 콘텐츠, 마스터의 코드 리뷰, 실시간 커뮤니케이션 등이 지속적으로 제공됩니다.
코드투게더 과정을 통해 스스로 소프트웨어를 학습하는 방법을 배우고, 동료와 함께 성장할 수 있습니다.

  • 자바스크립트 Part1

  • 클린 리액트 웹프로그래밍

    React는 가장 인기있는 웹프론트엔드 프레임워크입니다. 하지만 React를 제대로 이해하고 사용하는 것은 결코 쉽지 않습니다. React를 다루기 전에 알아야 할 핵심개념들을 잘 알고 시작하는 것이 중요합니다. 그리고 React 웹프로그래밍의 렌더링 원리와 디자인패턴, Best practice를 알아둘 필요가 있습니다.
